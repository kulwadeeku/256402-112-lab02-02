## Homework 2-1
Write a program Quadratic.java that takes three numbers (a, b, c) from a command line 
and solves for the roots of a quadratic equation: **ax<sup>2</sup> + bx + c**.

### Sample Run 1:
```
$ java Quadratic 1.0 -3.0 2.0
2.0
1.0
```

### Sample Run 2:
```
$ java Quadratic 1.0 -1.0 -1.0
1.618033988749895
-0.6180339887498949
```

### Sample Run 3:
```
$ java Quadratic 1.0 1.0 1.0
NaN
NaN
```
